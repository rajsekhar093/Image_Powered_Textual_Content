Project Overview

Image Caption Generator project, a fusion of computer vision and natural language processing that delivers insightful and descriptive captions for images. In a world dominated by visual content, our system stands out by not only recognizing and tagging images but by generating human-like captions that add context and meaning to the visual elements.

This project employs a synergistic combination of ResNet(Residual Neural Network) for image feature extraction and LSTM (Long Short-Term Memory) for sequential language modeling. Furthermore, our project introduces a unique dimension by incorporating voice for captioning. The integration of voice as a supplementary modality adds an additional layer of richness to the generated captions, enhancing the overall user experience and accessibility of the system.

Dataset

Here we use Flickr8k dataset. The Flickr8k dataset has become a standard benchmark for sentence-based image description for beginners.Flickr 8k dataset contains 8092 images and up to five captions for each image. Such annotations are essential for continued progress in automatic image description and grounded language understanding.

They enable us to define a new benchmark for localization of textual entity mentions in an image.
